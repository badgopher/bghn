The database

Like the vast majority of php sites developed by Internet 1.0 professional webmasters, the database runs on MySQL 5-something. The source .zip file contains 2 SQL scripts -- a schema-only script and a schema & data script. I included the one with data to help give a sense of how the site works (see earlier comment regarding documentation).

Initial setup

Create the database. Edit dbStringBuilder.php and enter the specifics for your database connection then run the script. Take the resulting string and update the DB_CONNECT definition in inc.config.php.

Cron?

Cron. There are 2 jobs that make the site tick:

*/15 * * * * /path/to/php "/path/to/webroot/parser/import.php"

0 0 * * 0 /path/to/php "/path/to/webroot/dbcleanup.php"

The first one runs the process that updates the feeds every 15 minutes. The second one keeps the database from growing into an unwieldy mess.
